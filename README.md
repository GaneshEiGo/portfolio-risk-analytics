# Global Portfolio Risk & Performance Analytics System

## Overview
This project analyzes a diversified portfolio of Indian and US equities using historical market data. The objective is to evaluate risk-return tradeoffs and identify the optimal portfolio allocation using Monte Carlo simulation.

## Problem Statement
Investors must balance expected return with portfolio risk. This project simulates 10,000 portfolio combinations to determine the most efficient allocation based on Sharpe Ratio optimization.

## Methodology
- Collected 5 years of historical price data
- Computed daily and annualized returns
- Built covariance matrix
- Simulated 10,000 portfolios
- Calculated Expected Return, Volatility, and Sharpe Ratio
- Identified Maximum Sharpe Portfolio
- Generated capital allocation for ₹10,00,000 investment

## Tools Used
- Python (Pandas, NumPy)
- Monte Carlo Simulation
- Power BI
